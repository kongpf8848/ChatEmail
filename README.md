## 使用

### 初始化向量数据库
[Create a vector db on Supabase](doc/vectordb/supabase.md)

### 设置环境变量
1. 在代码根目录创建.env文件
2. 参考文档[doc/env-vars.md](doc/env-vars.md)设置.env文件中的环境变量

## 本地运行
执行以下命令：
```shell
npm install

npm run dev
```
打开浏览器，访问[http://localhost:3000](http://localhost:300)
